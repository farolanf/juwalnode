# Turing Full Stack Challenge

## Requirements

- Node >= 10.0.0 for full ES 2018 support

## Setup

### Backend

- configure mysql in `server/config/development.js`
- `npm install` or `yarn`
- `node cmd data:initdb` to initialize and populate database
- `npm start`

### Frontend

- `npm install` or `yarn`
- `npm start`
